# Dossier Logo

Placez vos fichiers de logo dans ce dossier.

## Formats supportés

Le composant Header cherche automatiquement les fichiers suivants (dans cet ordre) :
- `logo.png`
- `Logo.png`
- `logo.svg`
- `Logo.svg`
- `logo-black.png`
- `logo-white.png`

## Recommandations

- **Format PNG** : Pour les logos avec transparence
- **Format SVG** : Pour les logos vectoriels (recommandé pour la qualité)
- **Taille** : Le logo sera automatiquement redimensionné à 40px de hauteur
- **Couleur** : Noir sur fond blanc (conforme au thème minimaliste)

## Fallback

Si aucun logo n'est trouvé, le texte "LA GRAVISTERIE" sera affiché à la place.
